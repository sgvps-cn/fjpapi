# 飞机盘解析工具

一个轻量级、美观易用的飞机盘分享链接解析工具，基于 HTML + Tailwind CSS 开发，无需后端部署，纯前端运行。

![License](https://img.shields.io/badge/license-MIT-green)
![Language](https://img.shields.io/badge/language-HTML/CSS/JS-blue)
![UI](https://img.shields.io/badge/UI-Tailwind%20CSS-purple)

## 🌟 功能特点

- 🎨 现代化 UI 设计，响应式布局，适配手机/电脑等多设备
- ⚡ 一键解析飞机盘分享链接，快速获取文件信息和下载地址
- 📋 便捷的下载链接复制功能，支持一键复制到剪贴板
- 🚨 完善的加载状态和错误提示，提升用户体验
- 🚀 无后端依赖，纯前端运行，部署简单
